# AI Post Generator

AI 驅動的病毒式貼文生成器，內建從 Reddit、Twitter/X、Dcard、PTT、Facebook 提取的高讚公式模板。

## 功能

- 🎯 高讚公式模板庫（按讚數排序）
- 📝 一鍵生成自訂貼文
- 🎨 Midjourney 配圖提示詞產生器
- 💡 互動技巧選擇器
- 📊 支援 6 大平台：Reddit、Twitter/X、Dcard、PTT、Facebook、通用

## 部署到 Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/YOUR_USERNAME/ai-post-generator)

或手動部署：

```bash
npm install -g vercel
cd ai-post-generator
vercel
```

## 本地開發

```bash
npm install
npm run dev
```
